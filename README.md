## Deployment

### Podman Compose

```yaml
services:
  stump:
    image: ghcr.io/daemonless/stump:latest
    container_name: stump
    environment:
      - PUID=1000
      - PGID=1000
      - TZ=UTC
    volumes:
      - "/path/to/containers/stump:/config"
      - "/path/to/containers/stump/data:/data" # optional
    ports:
      - 10801:10801
    restart: unless-stopped
```
